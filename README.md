### ⚡ What I Do
- Build structured backend APIs with **NestJS**, **TypeScript**, and **Node.js**.
- Design relational data models using **PostgreSQL** and **TypeORM**.
- Implement authentication and authorization flows with **JWT** and **bcrypt**.
- Containerize backend services with **Docker** and **Docker Compose**.
- Keep code maintainable with **ESLint**, **Prettier**, and automated testing with **Jest**.
